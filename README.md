# Energy-Consumption-Prediction-Using-FastAPI-and-PCA
A Machine Learning web application built with FastAPI that predicts energy consumption using PCA (Principal Component Analysis) and a Random Forest Regressor. The project includes data preprocessing, dimensionality reduction, interactive visualizations, and a prediction interface.
# Energy Consumption Prediction Dashboard
## Project Description
This project predicts energy consumption using a Machine Learning model trained with PCA and Random Forest.
## Features
- Home Page
- Dashboard with EDA charts
- Energy Consumption Prediction
- PCA Dimensionality Reduction
## Technologies
- Python
- FastAPI
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Joblib
## Run the Project
Install dependencies:
```bash
pip install -r requirements.txt
```
Run:
```bash
uvicorn main:app --reload
```
Open:
```
http://127.0.0.1:8000
```
